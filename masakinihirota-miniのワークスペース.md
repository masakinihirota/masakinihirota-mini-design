# masakinihirota-miniのワークスペース

3つのフォルダをVSCodeのワークスペースに追加しています。
ワークスペースへの追加方法は、VSCodeを開いて、ファイルメニューから「ワークスペースを開く」を選択し、フォルダをそれぞれ選択します。

1. masakinihirota-mini
Webアプリのコード

2. masakinihirota-mini-design
構想、概要、要件定義、仕様書、設計書、仕様書、実装手順
などのドキュメント

3. masakinihirota-mini-docs
Webアプリのドキュメント
vitepressで作成したドキュメント

# Clineでの開発

masakinihirota-mini-design をClineに読み込ませます。
Webアプリのコードは masakinihirota-mini に出力してもらいます。
作ったWebアプリのドキュメントは masakinihirota-mini-docs に出力してもらいます。

これができるかどうかを試します。
